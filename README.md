# Vocab Master Bot

Telegram-бот для изучения английского и корейского языков.
Функции:
- Приветствие нового пользователя
- Автоматическое определение языка (RU/EN)
- Перевод слов на английский и корейский
- Добавление и удаление слов из словаря
- Просмотр словаря

## Установка
pip install -r requirements.txt

## Запуск
python bot.py
